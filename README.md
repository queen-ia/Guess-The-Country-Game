## Project Overview
**A Java-based console game where players guess a hidden country based on distance clues, using both solo and multiplayer modes. 
It incorporates file input and is a structured gameplay.**

## How to Compile and Run

### Prerequisites
* Java Development Kit (JDK) 8 or higher installed
* A valid `.tsv` data file containing the country information

### Compile and Run

Open your terminal or command prompt, navigate to the directory containing the source file, and run:

```bash
javac GuessTheCountryGame.java
java GuessTheCountryGame countries.tsv
```
